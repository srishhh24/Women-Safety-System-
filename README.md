# Women-Safety-System-

A web-based Women Safety System designed to enhance personal security using modern web technologies. This project provides quick access to emergency features like sending alerts, sharing location, and triggering safety actions in critical situations.

---

## 🌟 Features

* 🚨 **Emergency SOS Button** – Instantly trigger alerts
* 📍 **Live Location Sharing** (via browser geolocation)
* 📞 **Quick Emergency Contacts Access**
* 🔊 **Alarm System** – Loud alert sound for nearby attention
* 📱 Responsive design (works on mobile & desktop)

---

## 🧰 Tech Stack

* HTML5
* CSS3
* JavaScript
